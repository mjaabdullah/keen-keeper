# KeenKeeper — Keep Your Friendships Alive

A personal friend manager that helps to stay connected with the people who matter most. Tracks when last contacted each friend, and keeps a history of calls, texts, and videos.

## Technologies Used

- React
- Vite
- React Router
- Tailwind CSS
- daisyUI
- Recharts
- React Toast

## Key Features

1. **Friend Status Tracker** — Cards show whether overdue, almost due, or on track with each friend.
3. **Friend Details** - When click on a card, detailed information about it is displayed.
2. **Quick Check-In** — Log a Call, Text, or Video interaction in one click, which instantly saves to timeline.
3. **Friendship Analytics** — A pie chart showing the breakdown of interactions across calls, texts, and video chats.
